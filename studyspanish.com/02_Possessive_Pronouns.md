# Possessive Pronouns

## Adjective vs. Pronoun

| Type      | Function         | Example             |
| --------- | ---------------- | ------------------- |
| Adjective | Describes a noun | *My book is large.* |
| Pronoun   | Replaces a noun  | *Yours is small.*   |
