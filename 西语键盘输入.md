# 西语键盘输入

Windows 10 系统下，从自带输入法中选择 `西班牙语（西班牙）` 键盘，最好不要选择其他国家。

![西班牙语（西班牙）键盘](images/西班牙语(西班牙)键盘.jpg)

## 特殊字符输入

- `á`：`'` 键 + `a` , `Á`：`'` 键 + `Shift` + `a`
- `é`：`'` 键 + `e` , `É`：`'` 键 + `Shift` + `e`
- `í`：`'` 键 + `i` , `Í`：`'` 键 + `Shift` + `i`
- `ó`：`'` 键 + `o` , `Ó`：`'` 键 + `Shift` + `o`
- `ú`：`'` 键 + `u` , `Ú`：`'` 键 + `Shift` + `u`
- `ü`：`'` 键 + `u` , `Ü`：`'` 键 + `Shift` + `u`
- `ñ`：`;` 键 , `Ñ`：`;` 键 + `Shift`

## 标点符号

- `,`：`,` 键
- `.`：`.` 键
- `:`：`Shift` + `.`
- `;`：`Shift` + `,`
- `¿`：`Shift` + `=`
- `?`：`Shift` + `-`
- `¡`：`=` 键
- `!`：`Shift` + `1`
- `(`: `Shift` + `8`
- `)`: `Shift` + `9`
- `[`: 右 `Alt` + `[`
- `]`: 右 `Alt` + `]`
- `{`: 右 `Alt` + `'`
- `}`: 右 `Alt` + `\`
- `+`: `]` 键
- `-`: `/` 键
- `*`: `Shift` + `]`
- `/`: `Shift` + `7`
- `=`: `Shift` + `0`
- `%`: `Shift` + `5`
